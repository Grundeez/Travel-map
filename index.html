⚙ GitHub Auto-Save

Connect to GitHub so every place you add or edit saves automatically to your index.html — no manual uploading ever again.

1
Go to github.com/settings/tokens/new — log in if needed.
2
Give it any name (e.g. travel-map). Under Scopes, tick repo. Scroll down and click Generate token.
3
Copy the token (starts with ghp_) and paste it below. It's stored only in your browser.
